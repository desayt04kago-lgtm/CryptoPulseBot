🚀 CryptoPulseBot

Telegram-бот для отслеживания изменений цен криптовалют с персональными уведомлениями.

📋 Описание

Бот парсит актуальные цены криптовалют с CoinMarketCap и отправляет уведомления пользователям, когда цена выбранной монеты изменяется на заданный процент.

✨ Возможности

📊 Просмотр всех криптовалют с текущими ценами

🔔 Персональные уведомления при изменении цены

⚙️ Настройка порога срабатывания (от 5% до 1000%)

✅ Подписка/отписка от любых монет в один клик

👀Отслеживание роста и падения цен

🕐 Автоматическая проверка каждые 60 минут

🔇 Включение/отключение уведомлений


📁 Структура проекта

    CryptoPulseBot/
    ├── main.py              # Основной файл бота (handlers, меню, уведомления)

    ├── .env                 # Переменные окружения (токен, пароль от БД)

    └── README.md            # Документация

🛠 Установка

1. Клонируйте репозиторий

        git clone https://github.com/ваш-username/CryptoPulseBot.git
        cd CryptoPulseBot

2. Создайте виртуальное окружение

        python -m venv .venv
        .venv\Scripts\activate  # Windows
        # или
        source .venv/bin/activate  # Linux/Mac

3. Установите зависимости

        pip install -r requirements.txt

4. Создайте файл .env

        tg_bot_token=ваш_токен_от_BotFather
        database_password=пароль_от_PostgreSQL

5. Запустите бота
    
        python main.py

🎮 Как использовать
1. Запуск бота

        Отправьте /start или любое сообщение боту.
2. Регистрация

        Бот предложит зарегистрироваться и установить порог уведомления (5-1000%).
3. Подписка на монеты

        Нажмите «Криптовалюта» в меню
        Выберите монеты кнопками (✅ — подписан, ❌ — не подписан)
4. Просмотр подписок

        Нажмите «Подписки» — увидите все активные подписки с возможностью отписаться.
5. Настройки

        Нажмите «Настройки» для:
        Изменения процента уведомления
        Включения/отключения рассылки

🔧 Команды бота

    Команда           Описание
    /start            Запуск бота и регистрация
    Меню              Главное меню
    Криптовалюта      Список всех монет для подписки
    Подписки          Активные подписки пользователя
    Настройки         Изменение процента и статуса уведомлений
    Поменять процент  Установка нового порога
    вкл/выкл рассылку Управление уведомлениями
